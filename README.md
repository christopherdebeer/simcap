# SIMCAP Data Branch

This branch contains session data files for the SIMCAP project.

## Structure

```
GAMBIT/
  └── {timestamp}.json    # Session data files
```

## Usage

Session data is uploaded here automatically via the web collector.

Files are served via `raw.githubusercontent.com`:
```
https://raw.githubusercontent.com/christopherdebeer/simcap/data/GAMBIT/{filename}
```

## Note

This branch only contains data files. Source code is on the `main` branch.
