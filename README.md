## Components

### [GAMBIT](src/web/GAMBIT/)
**Gyroscope Accelerometer Magnetometer Baseline Inference Telemetry**

Web UI for baseline data collection via BLE. Active implementation with real-time sensor visualization.

`Status: Active`

### [P0](src/web/P0/)
**Initial Prototype**

WebSocket-based data visualization and capture interface.

`Status: Prototype`

### [Documentation](docs/)
**Conceptual Analysis & Technical Docs**

Vision vs. reality analysis and roadmap forward.

`Status: Documentation`

### [JOYPAD](src/web/JOYPAD/)
**Dual-Hand Game Controller Concept**

Emulating controller HID from sensor data using two SIMCAP devices.

`Status: Concept`

### [FFO$$](src/web/FFO$$/)
**Fist Full Of Dollars**

$1 family algorithms for gesture inference from low-dimensional observation.

`Status: Research`
